# Source Citation Standards

## ECF Document Citations

Format: `ECF Doc. [NUMBER], filed [DATE], p. [PAGE]`

When hosted PDF available, use hyperlink:
`[ECF Doc. 1328-44](../sources/giuffre-v-maxwell/ecf-1328-44.pdf)`
